## ğŸ’¡ Tips

### For Developers
- Always test language switcher on all pages
- Use browser DevTools to test responsive
- Check console for animation errors
- Test with `prefers-reduced-motion` enabled

### For Designers
- Design mobile-first (375px baseline)
- Test on real devices, not just DevTools
- Use consistent spacing (4, 8, 12, 16, 24)
- Follow cyan/teal color scheme

### For Content Writers
- Update both `id.json` and `en.json`
- Keep translations consistent
- Use clear, concise language
- Test language switcher after changes
