## ğŸ¯ Key Design Patterns Identified

### 1. **Alternating Layout Pattern** â­ MOST IMPORTANT
**Location:** Screenshot 5 & 6
**Pattern:** Content tidak dalam grid monoton, tapi alternating left-right
**Why It Works:**
- Creates visual rhythm dan flow
- Mengurangi monotony
- More engaging untuk scroll experience
- Setiap item feels unique

**Application ke Impact Case Studies:**
- Case study 1: Image left, content right
- Case study 2: Content left, image right
- Case study 3: Image left, content right
- Creates zig-zag visual flow

---

### 2. **Large Imagery as Hero**
**Pattern:** Images bukan small thumbnails, tapi prominent/large
**Sizes:** Minimum 50% width pada desktop, full-width pada mobile
**Treatment:**
- High-quality images
- Subtle overlays untuk text readability
- Images sebagai focal point, bukan afterthought

---

### 3. **Numbers/Stats as Visual Elements**
**Pattern:** Large, bold numbers sebagai design element
**Implementation:**
- Font size: 2x-3x body text
- Bold atau extra-bold weight
- Accent colors untuk highlight
- Surrounded by whitespace

---

### 4. **Clean White/Light Backgrounds**
**Pattern:** Majority of sections use white/light backgrounds
**Benefits:**
- Professional, trustworthy feel
- Better readability
- Content sebagai hero, bukan background
- Works well dengan colorful accents

---

### 5. **Subtle Animations & Transitions**
**Pattern:** Smooth transitions, tidak jarring
**Types:**
- Fade-in pada scroll
- Scale effects pada hover
- Smooth color transitions
- Parallax effects (subtle)

## ğŸ“ Specific Design Elements to Implement

### 1. **Typography Hierarchy**
```
Brand Name: 48-64px, Bold, Dark
ER Score: 72-96px, Black, Accent Color
Result: 24-32px, Semibold
Description: 16-18px, Regular, Gray
Strategy: 14-16px, Medium, Bullet points
```

### 2. **Color Usage**
- Background: White (bg-white) atau light gray (bg-gray-50)
- Accent: Primary gradient untuk stats/buttons
- Text: Gray-900 untuk headings, Gray-600 untuk body
- Overlays: Black/60 untuk text on images

### 3. **Spacing**
- Section padding: py-24 (6rem) atau py-32 (8rem)
- Gap antar elements: gap-8 atau gap-12
- Card padding: p-8 atau p-10
- Whitespace: Generous, tidak cramped

### 4. **Image Treatment**
- Width: 50% pada desktop, 100% pada mobile
- Height: Auto atau aspect ratio 4:3
- Overlay: Gradient from-black/60 via-transparent to-black/60
- Border radius: rounded-2xl atau rounded-3xl
- Shadow: shadow-2xl untuk depth
